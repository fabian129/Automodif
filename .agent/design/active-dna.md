# Active Design DNA

## Brand Identity
- **Name:** Automodif
- **Core Value:** Professional, Trustworthy, Modern, Service-Oriented
- **Voice:** Knowledgeable, Transparent, Welcoming

## Color Palette
- **Primary:** `#0052CC` (Automodif Blue) - Used for primary actions, buttons, and key highlights.
- **Secondary:** `#0A0A0A` (Dark Surface) - Used for cards and secondary backgrounds.
- **Background:** `#030303` (Almost Black) - Main background color.
- **Text:**
  - Headings: `#FFFFFF`
  - Body: `#D4D4D8` (Zinc-300)
  - Muted: `#A1A1AA` (Zinc-400)
- **Accent:** `#0052CC` - Used for subtle glows and borders.
- **Status:**
  - Success: `#22c55e`
  - Error: `#ef4444`

## Typography
- **Font Family:** Inter (Sans Serif)
- **Weights:** Light (300), Regular (400), Medium (500), Semibold (600)

## Spacing values
- Standard grid: 4px base unit.
- Section padding: `py-24` (96px).
- Container padding: `px-6`.

## Effects
- **Glassmorphism:** `bg-white/5 backdrop-blur-md border border-white/10`
- **Glow:** Radial gradients for hover effects.
- **Shadows:** Subtle colored shadows depending on context (e.g., primary color glow).

## Motion DNA
- **Transitions:** `duration-300 ease-in-out`
- **Scroll Reveals:** Staggered fade-in-up animations (`y: 20 -> y: 0`, `opacity: 0 -> 1`).
- **Hover:** Slight scale or distinct border color change.
- **Marquee:** Continuous linear scrolling.
